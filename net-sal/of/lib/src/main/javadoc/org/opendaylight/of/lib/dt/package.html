A set of classes representing data values used in the OpenFlow protocol.

These classes are based on the "Cacheable Data Type" design and provide the following benefits:

  • Type-safety in API definitions
  • Instances are immutable (and thus inherently threadsafe)
  • Instances are Serializable so can be passed from VM to VM
  • equals() and hashCode() have been overridden to ensure instances play nicely with the Collection framework
  • The Comparable interface has been implemented to provide an "intuitive" sort order
  • valueOf() static methods allow for creation of instances from a variety of "external" representations
  • toType() instance methods convert back to "external" representations
  • Under-the-covers memory footprint optimizations

Collection of facilities to translate a set of command-line arguments into a map of name-to-object bindings, using an XML-encoded set of syntax definitions.

The {@link org.opendaylight.util.syntax.SyntaxCompiler syntax compiler} generates a {@link org.opendaylight.util.syntax.SyntaxPackage syntax package}, which akin to a compiled regular expression, can be used by a {@link org.opendaylight.util.syntax.SyntaxMatcher syntax matcher} to parse and validate arbitrary command-line arguments into a {@link org.opendaylight.util.syntax.Parameters map} of named {@link java.io.Serializable serializable} {@link org.opendaylight.util.syntax.Parameters parameters}.

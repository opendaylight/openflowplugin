Network packet parsing library.
  • Packet buffer reader and writer utilities
  • Representations of network packet structures
  • Network protocol decoders and encoders
This is a comprehensive list of the protocols supported and the OpenFlow related match fields:

Outermost layer:

  • Ethernet - Ethernet2, Ethernet 802.3, VLAN ID (802.1Q)
    • destination address
    • Ethernet source address
    • Ethernet frame type
    • VLAN ID
    • VLAN priority

Next layer group:

  • MPLS
    • MPLS label
    • MPLS TTL
    • MPLS Bottom of Label Stack bit
  • ARP
    • opcode
    • source IPv4 address
    • target IPv4 address
    • source hardware address
    • target hardware address
  • IP
    • IPv4 TOS Differentiated Services Field Codepoints (DSFC)
    • IPv4 TOS Explicit Congestion Notification (ECN)
    • IPv4 protocol type
    • IPv4 source address
    • IPv4 destination address
  • IPv6
    • IPv6 source address
    • IPv6 destination address
    • IPv6 flow label
  • LLDP
  • ICMP
    • ICMPv4 type
    • ICMPv4 code
  • ICMPV6
    • ICMPv6 type
    • ICMPv6 code
    • target address for Neighbor Discovery
    • source link-layer for Neighbor Discovery
    • target link-layer for Neighbor Discovery
Next layer group:
  • TCP
    • TCP source port
    • TCP destination port
  • SCTP
    • SCTP source port
    • SCTP destination port
  • UDP
    • UDP source port
    • UDP destination port
Innermost layer group:
  • DHCP
